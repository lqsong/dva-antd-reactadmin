<h1>
dva-antd-reactadmin
    <h3>React 16.*.* admin management system template based on (Dva and Antd).</h3>
</h1>

## Getting started
```bush
# clone the project
git clone https://github.com/lqsong/dva-antd-reactadmin.git

// install dependencies
npm install

// develop
npm run start
```

## Build
```bush
npm run build
```
